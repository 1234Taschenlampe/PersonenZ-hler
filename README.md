# PersonenZähler

Animierte Konzeptseite für ein KI-Kameraprojekt der Stadt Wangen.

Die Datei `index.html` beschreibt das geplante Zwei-Türme-System mit Kamera,
Licht, ESP32, Raspberry Pi 5 mit 16 GB RAM, lokaler KI-Auswertung und zusätzlichen
Sensorvarianten wie Druckplatte, LIDAR und Lichtschranke.
